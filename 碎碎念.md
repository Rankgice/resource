**数据库 CDC（Change Data Capture）就是捕获数据库中的数据变更（增、删、改），并实时同步到其他系统的技术。**

例如：

```
MySQL 数据变化
      ↓
   CDC 捕获 binlog
      ↓
 Kafka / Flink / ES / 数据仓库
      ↓
实时同步、分析、搜索
```

常见实现：

* MySQL → 读取 **binlog**
* PostgreSQL → 读取 **WAL**
* Oracle → 读取 **Redo Log**

常用工具：

* Debezium
* Canal
* Flink CDC

一句话理解：**CDC 就像数据库的“监听器”，谁改了数据，它就把变化事件通知出去。**
